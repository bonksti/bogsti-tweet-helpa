# Privacy Policy for bogsti tweet helpa

**Last Updated: November 2025**

bogsti tweet helpa (“the extension”) is designed to summarize the content of tweets that users intentionally choose to summarize. We value your privacy and are committed to protecting it. This Privacy Policy explains what data we access, how we use it, and what we do not collect.

---

## 1. Information We Collect
bogsti tweet helpa does **not** collect, store, sell, share, or transfer any personally identifiable information or user data.

The extension does not collect:
- Names or contact information  
- Location or IP data  
- Browsing history  
- Authentication information  
- Financial information  
- Personal communications  
- User-generated content  
- Sensitive or protected data  

We do not use cookies, analytics, trackers, or fingerprinting technologies.

---

## 2. What the Extension Accesses
The extension accesses only the content of the tweet currently open in your browser, **and only when you explicitly activate the summarization feature**.

Specifically:
- The visible tweet text  
- Publicly displayed tweet metadata (if present)

This information is used **solely** to create a summary at your request.

No background data collection occurs.

---

## 3. Use of OpenAI API
To generate summaries, the extension sends the tweet text **you choose** to OpenAI’s API using your personal API key.

- The extension does **not** send any additional data.  
- You control when and what gets sent.  
- Your API key is stored locally using Chrome’s sync storage and is never transmitted anywhere else.

OpenAI’s processing of data is governed by their own privacy policy.

---

## 4. Local Storage
The extension stores only:
- Your OpenAI API key  
- Optional UI settings (theme, preferences)

These are stored locally in Chrome sync storage and are **never** transmitted or collected by us.

No summary, tweet text, or metadata is stored.

---

## 5. No Data Sharing
We do not sell, share, or transfer any data to third parties.

We do not participate in any advertising, tracking, analytics, or profiling systems.

---

## 6. Permissions Explanation

### activeTab  
Allows reading the visible tweet content only after user action.

### scripting  
Used to extract tweet text and insert summaries back into the page.

### storage  
Stores only the user’s API key and customization settings.

### Host Permissions (x.com, twitter.com, api.openai.com)  
Required to:
- Read the tweet text on x.com/twitter.com  
- Send user-selected text to the OpenAI API for summarization

These permissions serve the extension’s **single purpose** and nothing else.

---

## 7. Children’s Privacy
The extension is not intended for use by children under 13. We do not knowingly collect data from children.
